Famili tree

A group project where we created a website for making, editing, managing, and exporting family trees. I was responsible for authorization, authentication, uploading files, and sending files to the server. What I’ve learned: Security authorization and authentication using JWT tokens. Creating, throwing, and handling own exceptions with custom middleware. Working with DTOs.

Train speed calibration

Application to calibrate the speedometer by adjusting for changes in wheel size on trains. During the operation of trains, the wheel circumference decreases. This app uses GPS data to calibrate the circumference, ensuring the speedometer remains accurate even in situations where GPS access is not available. What I’ve learned: Using dev containers in VSC, multithreading in C++

Moodle++

A copy of a well-known service called Moodle. It includes three user types, course management, user management, and file sending. What I’ve learned: React Router, private routes based on roles, serving props with context, bootstrap, yup.
